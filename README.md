# Leo Club Website

## Features

- **Membership Application System** - Public form for submitting membership applications with personal details, education information, and application reasons
- **Contact Form** - Public contact form for visitors to send messages and inquiries
- **Admin Panel** - Secure admin dashboard with session-based authentication
- **Member Management** - Admin can view, approve/reject, and delete membership applications
- **Contact Message Management** - Admin can view and delete contact messages
- **Dashboard** - Admin dashboard with overview statistics
- **Website Pages** - Multiple pages including Home, About, Board Members, Gallery, Events, Awards, Achievements, Membership, and Contact

## Tools & Technologies

### Backend
- **Node.js** - Runtime environment
- **Express.js** - Web application framework
- **MongoDB** - Database
- **Mongoose** - MongoDB object modeling
- **Express-session** - Session management
- **CORS** - Cross-origin resource sharing
- **Body-parser** - Request body parsing
- **dotenv** - Environment variable management
- **Nodemon** - Development server auto-reload

### Frontend
- **HTML** - Markup language
- **CSS** - Styling
- **JavaScript** - Client-side scripting
